## 二、技术架构

### 后端架构

```
┌─────────────────────────────────────────────────────────┐
│                    FastAPI 应用                          │
├─────────────────────────────────────────────────────────┤
│  API 路由                                                │
│  ├── /api/v1/auth/*    (认证)                           │
│  ├── /api/v1/schedule/* (日程)                          │
│  ├── /api/v1/parse/*   (解析)                           │
│  ├── /api/v1/qa/*      (问答)                           │
│  └── /api/admin/*      (管理后台)                       │
├─────────────────────────────────────────────────────────┤
│  核心模块                                                │
│  ├── auth.py          (JWT 认证)                        │
│  ├── llm.py           (Qwen LLM)                        │
│  ├── rag.py           (RAG 检索)                        │
│  ├── parser.py        (信息解析)                        │
│  ├── scheduler.py     (日程管理)                        │
│  └── reminder.py      (提醒服务)                        │
├─────────────────────────────────────────────────────────┤
│  数据层                                                  │
│  ├── SQLAlchemy ORM                                      │
│  ├── MySQL / SQLite                                      │
│  └── Qdrant (向量数据库)                                 │
└─────────────────────────────────────────────────────────┘
```

## 五、API 安全

- 权限控制：管理员/普通用户

### API 权限

|----------|----------|
| `/api/v1/*` | 公开 |
| `/api/v1/auth/me` | 登录用户 |
| `/api/admin/*` | 管理员 |

## 六、快速开始

### Windows 一键安装

```cmd
cd E:\Code\ai-test\AI-agent\campus_ai_secretary

REM 1. 安装
scripts\setup.bat

REM 2. 配置 .env 文件（填入 DASHSCOPE_API_KEY）

REM 3. 初始化数据库
scripts\init_db.bat

REM 4. 启动服务
scripts\run_demo.bat
```

### 访问地址

- **管理后台**: http://localhost:8000/admin
- **API 文档**: http://localhost:8000/docs
- **默认账号**: admin / admin123

## 七、数据库配置

### 使用 MySQL（推荐）

```env
DATABASE_URL=mysql+pymysql://root:123456@localhost:3306/campus_ai?charset=utf8mb4
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=123456
DB_NAME=campus_ai
```

### 使用 SQLite（无需配置）

系统会自动降级使用 SQLite，数据库文件：`campus_ai.db`

## 十、注意事项

1. **首次使用必须初始化数据库**
   ```bash
   python scripts/init_db.py
   ```

2. **MySQL 服务必须运行**（如使用 MySQL）

3. **DASHSCOPE_API_KEY 必填**（用于 LLM 功能）

4. **管理后台需要登录**（默认账号 admin/admin123）

5. **生产环境请修改默认密码**
